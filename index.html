pfc_marker

import PFClean timeline markers

Download this project as a tar.gz file Download Windows Installer only

outline

pfc_marker is a small windows tool to import standard cmx3600 edls, csv-textfiles or dvs clipster timeline markers into an existing pfclean sequence.
it works basically from left to right. buttons and selection are only activated when it makes a logical sense. all timecode calculations are defined by the selected pfclean-sequence.

edl (cmx3600): source- or record-tc only, no comments, dissolves are filtered
clipster-project: timeline-markers including names and comments
csv-text: one marker-event per line, separated by the first comma

compiled installer (msi) under /msi was tested with pfclean 2014-2015 on windows 7/8/10
current build under /build runs straight away

written in python3.4 and qt5, win64
