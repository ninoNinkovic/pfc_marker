pfc_marker

create PFClean timeline markers

Download this project as a .zip file Download this project as a tar.gz file

not yet released ...

pfc_marker is a small windows tool to import standard cmx3600 edls or dvs clipster timeline marker into an existing pfclean sequence.

tested with pfclean 2014-2015 on windows 7/8/10

written in python3.4 and qt5, win64

pfc_marker maintained by suminus
